# 1350. Excel Sheet Column Title

Difficulty: Easy

https://www.lintcode.com/problem/excel-sheet-column-title/description

**Description**  
Given a positive integer, return its corresponding column title as appear in an Excel sheet.
```
1 -> A
2 -> B
3 -> C
 ...
26 -> Z
27 -> AA
28 -> AB 
```

**Example**  
Example1
```
Input: 28
Output: "AB"
```
Example2
```
Input: 29
Output: "AC"
```
