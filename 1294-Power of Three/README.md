# 1294. Power of Three

Difficulty: Easy

https://www.lintcode.com/problem/power-of-three/description

**Description**  
Given an integer, write a function to determine if it is a power of three.

**Example**  
Example1
```
Input: n = 0
Output: False
```
Example2
```
Input: n = 9
Output: True
```

**Challenge**  
Could you do it without using any loop / recursion?
