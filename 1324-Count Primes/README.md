# 1324. Count Primes

Difficulty: Easy

https://www.lintcode.com/problem/count-primes/description

**Description**  
Count the number of prime numbers less than a non-negative number, n.

**Example**  
Example 1：
```
Input: n = 2
Output: 0
```
Example 2：
```
Input: n = 4
Output: 2
Explanation：2, 3 are prime number
```
