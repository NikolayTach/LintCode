# 772. Group Anagrams

Difficulty: Medium

http://www.lintcode.com/en/problem/group-anagrams/

Given an array of strings, group anagrams together.

**Notice**  
All inputs will be in lower-case.

**Example**  
```
Given strs = ["eat", "tea", "tan", "ate", "nat", "bat"],
Return 
[
    ["ate", "eat","tea"],
    ["nat","tan"],
    ["bat"]
]
```
