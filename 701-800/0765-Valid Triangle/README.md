# 765. Valid Triangle

Difficulty: Easy

http://www.lintcode.com/en/problem/valid-triangle/

Given three integers a, b, c, return true if they can form a triangle.

**Notice**  
What is Triangle? (Wikipedia)

**Example**  
```
Given a = 2, b = 3, c = 4
return true
Given a = 1, b = 2, c = 3
return false
```
