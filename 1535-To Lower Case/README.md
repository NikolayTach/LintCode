# 1535. To Lower Case

Difficulty: Easy

https://www.lintcode.com/problem/to-lower-case/description

**Description**  
Implement function ToLowerCase() that has a string parameter str, and returns the same string in lowercase.

**Example**  
Example 1:
```
Input: "Hello"
Output: "hello"
```
Example 2:
```
Input: "here"
Output: "here"
```
Example 3:
```
Input: "LOVELY"
Output: "lovely"
```
