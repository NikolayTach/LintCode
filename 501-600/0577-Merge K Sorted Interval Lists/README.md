# 577. Merge K Sorted Interval Lists

Difficulty: Medium

http://www.lintcode.com/en/problem/merge-k-sorted-interval-lists/

Merge K sorted interval lists into one sorted interval list. You need to merge overlapping intervals too.

**Example**  
Given
```
[
  [[1,3],[4,7],[6,8]],
  [[1,2],[9,10]]
]
```
Return
```
[[1,3],[4,8],[9,10]]
```
