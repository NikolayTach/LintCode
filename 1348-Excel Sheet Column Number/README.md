# 1348. Excel Sheet Column Number

Difficulty: Easy

https://www.lintcode.com/problem/excel-sheet-column-number/description

**Description**  
Given a column title as appear in an Excel sheet, return its corresponding column number.
```
A -> 1
B -> 2
C -> 3
...
Z -> 26
AA -> 27
AB -> 28
```

**Example**  
Example1
```
Input: "AB"
Output: 28
```
Example2
```
Input: "AC"
Output: 29
```
