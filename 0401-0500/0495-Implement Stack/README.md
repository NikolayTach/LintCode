# 495. Implement Stack

Difficulty: Easy

http://www.lintcode.com/en/problem/implement-stack/

Implement a stack. You can use any data structure inside a stack except stack itself to implement it.

**Example**  
```
push(1)
pop()
push(2)
top()  // return 2
pop()
isEmpty() // return true
push(3)
isEmpty() // return false
```
