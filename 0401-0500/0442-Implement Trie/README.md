# 442. Implement Tri

Difficulty: Medium

http://lintcode.com/en/problem/implement-trie/

Implement a trie with insert, search, and startsWith methods.

**Notice**  

You may assume that all inputs are consist of lowercase letters a-z.

**Example**  
``` 
insert("lintcode")
search("code")
>>> false
startsWith("lint")
>>> true
startsWith("linterror")
>>> false
insert("linterror")
search("lintcode)
>>> true
startsWith("linterror")
>>> true
```
