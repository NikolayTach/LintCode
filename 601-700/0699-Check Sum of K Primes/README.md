# 699. Check Sum of K Primes

Difficulty: Hard

http://www.lintcode.com/en/problem/check-sum-of-k-primes/

Given two numbers n and k. We need to find out if n can be written as sum of k prime numbers.

**Notice**  
* n <= 10 ^ 9
* Goldbach's conjecture

**Example**  
```
Given n = 10, k = 2
Return true   // 10 = 5 + 5

Given n = 2, k = 2
Return false
```
